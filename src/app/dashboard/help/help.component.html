Overview

—

The current version of this application allows you to keep track of your expenses and earning when conducting a business.

Features

  • Create, remove, and edit projects
  • Add expenses to specific project
  • View project revenue, cost, profit or loss in a pie chart

Instructions

—

To create new project:

  1. After logging in you are presented with your current projects and a form to add a new project.
  2. Fill out the form as follows:
    Type:
    Project type. (Only 'business' type in this version.)
    Title:
    The title of the project
    Budget:
    The customer's' initial payment or capital
    Estimate:
    The total that the client will pay for the project.
    Description:
    A short description of the project.
  3. Click on the 'Save' button.

To view the project details and edit the information:

  1. Click on the icon on the project list display
  2. Under the 'Overview' tab you can edit the project's information by clicking on the 'Edit' button, and the 'Save' button when done.
  3. The 'Expenses' tab contains the project's expenses. You can add a new expense by using the form as follows:
    Name:
    The name of the expense.
    Price:
    The price per unit of the expense.
    Qty:
    How many units were purchased.
  4. The 'Report' tab simply shows a visual representation of the project's' outcome.
